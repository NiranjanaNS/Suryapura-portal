# Design Explanation (~150 words)

I designed the Suryapura portal as a real Next.js + TypeScript + Tailwind
application rather than a static mockup, since the role calls for full
stack development skill, not just visual design. The palette pairs a deep
teal (agriculture, growth) with indigo (digital governance) and a marigold
accent tied to the "Suryapura" (Sun City) name — a sunrise motif recurs
across the hero illustration, stat cards, and footer to give the brand a
consistent thread. Typography uses Poppins for headings and Noto Sans
Devanagari for Hindi content, both loaded correctly for bilingual display.
Instead of generic stock photography, the hero uses a hand-built SVG
village illustration, while the gallery uses real, credited photographs.
The layout is fully responsive, built from reusable components (cards,
section headings, animated counters), with data centralized in one file —
so content can be swapped for a live API without touching any UI code.
